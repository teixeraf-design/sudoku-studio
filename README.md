# 🧩 Sudoku Studio

Gerador procedural de Sudoku 9x9 com estatísticas em tempo real e integração Firebase.

## ✨ Funcionalidades

- ✅ Geração procedural de tabuleiros válidos
- ✅ 5 níveis de dificuldade (Fácil a Gênio)
- ✅ Sistema de vidas (limite de erros)
- ✅ Timer com pausa
- ✅ Teclado virtual e suporte a teclado físico
- ✅ Login com Google (Firebase Auth)
- ✅ Estatísticas salvas no Firestore
- ✅ Histórico de partidas
- ✅ Modo Convidado (jogar sem login)
- ✅ PWA (instalável no celular)
- ✅ Responsivo (mobile/desktop)

## 🛠️ Tecnologias

- HTML5 / CSS3 / JavaScript (Vanilla)
- Firebase Authentication
- Firebase Firestore
- PWA (Service Worker)
